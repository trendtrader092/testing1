# Indian Market FVG Strategy (Fair Value Gap)

This folder contains screenshots and examples of a Fair Value Gap (FVG) trading strategy applied to Indian stock/index markets (primarily NIFTY). The setups are demonstrated across multiple timeframes, including 1-minute, 3-minute, and 5-minute charts.

---

## 📌 Strategy Overview

The Fair Value Gap (FVG) strategy is based on identifying price inefficiencies (gaps) created by strong bullish or bearish moves. Price often returns to these gaps before continuing in the original direction, offering potential entry opportunities.

This setup focuses on:
- Higher timeframe bias (3M + 5M)
- Execution on lower timeframe (1M)
- Liquidity and price imbalance zones

---

## 🧠 How the Strategy Works

1. **Identify Trend Direction**
   - Use 3-minute and 5-minute charts to determine overall market bias.

2. **Mark Fair Value Gaps (FVG)**
   - Locate imbalance zones where price moved aggressively leaving inefficient pricing.

3. **Wait for Retest**
   - Price often returns to fill the FVG before continuing trend direction.

4. **Entry on 1-Minute Chart**
   - Refine entries using the 1-minute timeframe for precision.

5. **Risk Management**
   - Always define stop-loss below/above the FVG zone or recent structure level.
   - Maintain proper risk-to-reward ratio (minimum 1:2 recommended).
